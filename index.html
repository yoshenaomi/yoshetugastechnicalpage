cara membuat tabel pada html

macam tag dalam membuat tabel:

dalam membuat tabel terdapat banyak sekali tag yang ada, yakni:

  • Tag table=/table untuk membungkus tabelnya
  • Tag thead=/thead untuk membungkus bagian kepala tabel
  • Tag tbody=/tbody untuk membungkus bagian body dari tabel
  • Tag tr=/tr (tabel row) untuk membuat baris
  • Tag td=/td (table data) untuk membuat sel
  • Tag th=/th (table head) untuk membuat judul pada header
